# steam-summer-sale-wallpaper

A small HTML/CSS/JS live wallpaper with a pixel-style clock and date using custom-made letters and digits.

https://github.com/user-attachments/assets/8394208e-9a31-4005-9cdb-e1c990e08fcb

## What it does
- Renders a background GIF and overlays a time/date display
- Uses pixel glyphs for digits and letters
- Exposes slider settings for positioning and sizes

## Run locally
- Open `index.html` in a browser
- Or load the folder in Wallpaper Engine (web wallpaper)
